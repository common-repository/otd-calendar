== Description ==

OTD Calendar shows a simple On-This-Day calendar.


== Installation ==

1. Upload 'OTDCalendar.php' to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to 'Widgets' section under 'Appearance' menu in WordPress and add the 'OTDCalendarWidget' in your sidebar.
